ทำไม Windows หรือ Chrome แจ้งเตือนว่าไฟล์นี้อันตราย?
-------------------------------------------------------
BoxBox ไม่ใช่ไวรัส แต่มีพฤติกรรมที่ระบบความปลอดภัยตรวจจับได้ เนื่องจาก:

  1. ไม่มี Code Signing Certificate
     ซอฟต์แวร์ที่ Windows เชื่อถือต้องซื้อใบรับรองจากบริษัทกลาง
     ราคาประมาณ 6,000–15,000 บาท/ปี ซึ่งผู้พัฒนาไม่มีงบประมาณส่วนนี้
     Windows SmartScreen จึงเตือนไฟล์ทุกตัวที่ไม่มีใบรับรอง

  2. ระบบอัปเดตอัตโนมัติ
     BoxBox ดาวน์โหลดและติดตั้งเวอร์ชันใหม่จาก Google Drive
     พฤติกรรมนี้คล้าย malware จึงถูก antivirus ตรวจจับ
     แต่ BoxBox ตรวจสอบ SHA-256 ก่อนรันทุกครั้ง ไม่รันไฟล์ที่ไม่ตรง

  3. ใช้ WebView2 + JavaScript Bridge
     สถาปัตยกรรมนี้ช่วยให้ UI ยืดหยุ่น แต่ antivirus บางตัว
     มองว่าเป็นเทคนิคที่ malware ใช้ด้วย

BoxBox ปลอดภัยหรือไม่?
------------------------
ปลอดภัยครับ — BoxBox ถูกพัฒนาเพื่อใช้ภายในโรงพยาบาลเท่านั้น
โค้ดทั้งหมดเขียนโดยเภสัชกรในกลุ่มงาน ไม่มีการส่งข้อมูลออกนอกองค์กร
นอกจาก Google Sheets ที่ใช้เก็บข้อมูลกล่องยา (ซึ่งผู้บริหารรับทราบ)

วิธีติดตั้งโดยไม่ให้ Windows บล็อก
-------------------------------------
  1. ถ้า Windows SmartScreen ขึ้น → คลิก "More info" → "Run anyway"
  2. ถ้า Chrome บล็อกการดาวน์โหลด → คลิกไอคอนลูกศรข้างไฟล์ → "Keep"
  3. ถ้า Antivirus บล็อก → ติดต่อ IT เพื่อเพิ่ม exclusion ใน Windows Defender:
       เส้นทาง: C:\Users\...\AppData\Local\Programs\BoxBox
